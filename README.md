# Web Tombola

Progetto realizzato come esercizio per Boolean Careers.

## Descrizione
Web Tombola è una semplice applicazione web che simula l’estrazione dei numeri della tombola, il classico gioco natalizio italiano. L’obiettivo è creare una griglia con i numeri da 1 a 90 e permettere l’estrazione casuale di ciascun numero, evidenziando quelli già usciti.

## Requisiti della consegna
- Generare dinamicamente la griglia dei numeri da 1 a 90.
- Implementare un pulsante per estrarre un numero casuale tra quelli non ancora usciti.
- Evidenziare i numeri estratti nella griglia.
- Impedire l’estrazione di numeri già usciti.
- Utilizzare HTML, CSS (anche con Bootstrap) e JavaScript vanilla.

## Struttura del progetto
- `index.html`: struttura della pagina.
- `css/style.css`: stili personalizzati.
- `js/scripts.js`: logica JavaScript per la tombola.

## Come usare il progetto
1. Clona la repository o scarica i file.
2. Apri `index.html` in un browser moderno.
3. Clicca sul pulsante "Estrai" per estrarre i numeri.
